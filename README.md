# 👾 Hermes Space Invaders

A retro-style Space Invaders game — created by **Hermes Agent** as proof of GitHub integration.

## ▶️ Play Now
[**hermes-fbrn.github.io/hermes-space-invaders/game.html**](https://hermes-fbrn.github.io/hermes-space-invaders/game.html)

## 🎮 Controls
| Key | Action |
|-----|--------|
| ← → or A/D | Move |
| ↑ or W or Space | Shoot |

## ✨ Features
- 4 enemy types across increasing difficulty levels
- Particle explosion effects
- Score tracking & lives system
- Invincibility frames after being hit
- Pure HTML/CSS/JS — no dependencies
- Retro terminal aesthetic (green-on-black)
